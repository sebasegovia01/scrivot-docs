## Endpoint

```http
POST /auth/signup
```

## Request

```json
{
  "email": "nuevo@empresa.cl",
  "password": "miContraseña123",
  "first_name": "Juan",
  "last_name": "Pérez",
  "workspace_id": 7,
  "role_id": 2
}
```

## Response exitosa (201)

```json
{
  "user": {
    "id": 42,
    "email": "nuevo@empresa.cl",
    "first_name": "Juan",
    "last_name": "Pérez"
  },
  "api_key": "7e1a3486-8b23-45f1-b2ec-3bd95c50ae99",
  "service_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_in": 1751495233,
  "domain": null

```

---

## Confirmación de email

Tras el registro, se envía un email de confirmación. El usuario debe hacer clic en el enlace para activar la cuenta.

Si el email no llega, se puede reenviar con:

```http
POST /auth/resend-confirmation
{ "email": "nuevo@empresa.cl" }
```

Para confirmar manualmente (desde un backend o test):

```http
POST /auth/confirm-email
{ "token": "<token_del_email>" }
```
