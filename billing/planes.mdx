## Límite de conversaciones (Free)

El plan Free tiene un límite de **30 conversaciones por mes calendario** (contadas en zona horaria America/Santiago).

- El límite se reinicia el **1 de cada mes**
- Las conversaciones activas no se cortan al alcanzar el límite
- Al alcanzar el límite, **no se permiten nuevas conversaciones** hasta el próximo mes o hasta actualizar el plan

Para verificar el uso actual:

```http
GET /billing/workspace/:workspaceId/usage
```

```json
{
  "allowed": false,
  "used": 30,
  "limit": 30,
  "plan": "free"
}
```

## Cambiar de plan

Para actualizar al plan Pro, ve a **Workspace → Billing → Suscribirse** y selecciona el intervalo de facturación (mensual o anual).

El pago se procesa a través de **MercadoPago**.
