---
title: Reset de contraseña
api: POST /auth/reset-password
description: Establecer nueva contraseña con el token del email
---

<ParamField body="token" type="string" required>
  Token recibido en el email de reset
</ParamField>

<ParamField body="password" type="string" required>
  Nueva contraseña
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.scrivot.cl/auth/reset-password \
  -H "Content-Type: application/json" \
  -d '{
    "token": "<token_del_email>",
    "password": "nuevaContraseña456"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Contraseña actualizada exitosamente"
}
```

```json 400
{
  "statusCode": 400,
  "message": "El token y la contraseña son obligatorios",
  "error": "Bad Request",
  "code": "MISSING_FIELDS"
}
```
</ResponseExample>
