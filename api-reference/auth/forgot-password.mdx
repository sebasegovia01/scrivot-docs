---
title: Recuperar contraseña
api: POST /auth/forgot-password
description: Solicitar email de reset de contraseña
---

<ParamField body="email" type="string" required>
  Correo electrónico de la cuenta
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.scrivot.cl/auth/forgot-password \
  -H "Content-Type: application/json" \
  -d '{"email": "usuario@empresa.cl"}'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Se envió un correo con instrucciones para restablecer tu contraseña"
}
```
</ResponseExample>
