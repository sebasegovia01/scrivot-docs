---
title: Inicio de sesión
api: POST /auth/login
description: Autenticarse y obtener un token de servicio
---

<ParamField body="email" type="string" required>
  Correo electrónico del usuario
</ParamField>

<ParamField body="password" type="string" required>
  Contraseña del usuario
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.scrivot.cl/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "usuario@empresa.cl", "password": "miContraseña123"}'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "service_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_in": 1751495233
}
```

```json 401
{
  "statusCode": 401,
  "message": "La contraseña es incorrecta",
  "error": "Unauthorized",
  "code": "INVALID_PASSWORD"
}
```

```json 404
{
  "statusCode": 404,
  "message": "El correo electrónico no está registrado",
  "error": "Not Found",
  "code": "EMAIL_NOT_REGISTERED"
}
```
</ResponseExample>
