---
title: Colecciones RAG
description: Gestionar colecciones de conocimiento
---

## Listar colecciones

```http
GET /rag/collections/workspace/{workspaceId}
Authorization: Bearer <token>
```

**Response:**
```json
[
  {
    "id": 1,
    "name": "FAQ General",
    "workspace_id": 7,
    "created_at": "2025-01-10T10:00:00.000Z"
  }
]
```

---

## Crear colección

```http
POST /rag/collections
Authorization: Bearer <token>
```

```json
{
  "name": "FAQ General",
  "workspace_id": 7,
  "description": "Preguntas frecuentes generales"
}
```

---

## Actualizar colección

```http
PUT /rag/collections/{id}
Authorization: Bearer <token>
```

---

## Eliminar colección

```http
DELETE /rag/collections/{id}
Authorization: Bearer <token>
```

<Warning>
  Eliminar una colección elimina también todos sus documentos y sus vectores.
</Warning>
