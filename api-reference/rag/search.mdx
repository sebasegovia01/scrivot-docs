## Búsqueda vectorial (semántica)

Busca documentos por similitud de significado.

```http
POST /rag/search
Authorization: Bearer <token>
```

```json
{
  "query": "¿cuánto cuesta el envío?",
  "collection_id": 1,
  "limit": 3,
  "format": "compact"
}
```

**Formatos de respuesta:**

| `format` | Descripción |
|---|---|
| `minimal` | Solo contenido y score (por defecto) |
| `compact` | Contenido + metadata básica |
| `full` | Documento completo |

**Response (`compact`):**
```json
[
  {
    "id": 42,
    "title": "Costos de envío",
    "content": "El envío estándar tiene un costo de $3.990...",
    "score": 0.92,
    "category": "Envíos"
  }
]
```

## Búsqueda por thread (widget)

Endpoint público usado por el widget para obtener contexto RAG sin autenticación de usuario:

```http
POST /rag/search/thread
```

```json
{
  "query": "horario de atención",
  "thread_id": "abc-123-def"
}
```

---

## Búsqueda híbrida

Combina similitud semántica con filtros de metadata:

```http
POST /rag/search/hybrid
Authorization: Bearer <token>
```

```json
{
  "query": "política de devoluciones",
  "collection_id": 1,
  "filters": {
    "category": "Soporte"
  },
  "limit": 5,
  "format": "compact"
}
```

---

## Búsqueda por metadata

Filtra documentos por campos específicos sin búsqueda semántica:

```http
GET /rag/documents/search?category=FAQ&collection_id=1
Authorization: Bearer <token>
```
