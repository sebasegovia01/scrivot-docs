---
title: Crear chatbot
api: POST /v1/chatbot
description: Crear un nuevo chatbot en el workspace
---

<Note>
  El `workspace_id` del body debe coincidir con el workspace del usuario autenticado.
</Note>

<ParamField body="workspace_id" type="number" required>
  ID del workspace
</ParamField>

<ParamField body="name" type="string" required>
  Nombre del chatbot (uso interno)
</ParamField>

<ParamField body="primary_language_id" type="number" required>
  ID del idioma principal
</ParamField>

<ParamField body="system_prompt" type="string">
  Instrucciones base del comportamiento del chatbot
</ParamField>

<RequestExample>
```bash cURL
curl -X POST https://api.scrivot.cl/v1/chatbot \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <service_token>" \
  -d '{
    "workspace_id": 7,
    "name": "Asistente de Ventas",
    "primary_language_id": 1,
    "system_prompt": "Eres el asistente de ventas de Acme Corp..."
  }'
```
</RequestExample>
