---
title: Eliminar chatbot
api: DELETE /v1/chatbot/{id}
description: Eliminar un chatbot del workspace
---

<ParamField path="id" type="number" required>
  ID del chatbot a eliminar
</ParamField>

<Warning>
  Esta acción es irreversible. Se eliminarán también los datos de configuración asociados.
</Warning>

<RequestExample>
```bash cURL
curl -X DELETE https://api.scrivot.cl/v1/chatbot/5 \
  -H "Authorization: Bearer <service_token>"
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Chatbot eliminado exitosamente"
}
```

```json 403
{
  "statusCode": 403,
  "message": "No tienes permisos para eliminar este chatbot",
  "error": "Forbidden",
  "code": "INSUFFICIENT_PERMISSIONS"
}
```
</ResponseExample>
