---
title: Actualizar chatbot
api: PUT /v1/chatbot/{id}
description: Modificar la configuración de un chatbot existente
---

<ParamField path="id" type="number" required>
  ID del chatbot
</ParamField>

<ParamField body="name" type="string">
  Nuevo nombre del chatbot
</ParamField>

<ParamField body="system_prompt" type="string">
  Nuevo system prompt
</ParamField>

<RequestExample>
```bash cURL
curl -X PUT https://api.scrivot.cl/v1/chatbot/5 \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <service_token>" \
  -d '{
    "name": "Asistente de Ventas v2",
    "system_prompt": "Instrucciones actualizadas..."
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 5,
  "name": "Asistente de Ventas v2",
  "workspace_id": 7,
  "updated_at": "2025-04-29T15:00:00.000Z"
}
```
</ResponseExample>
