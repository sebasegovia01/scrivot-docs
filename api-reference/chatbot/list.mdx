---
title: Listar chatbots
api: GET /v1/chatbot/workspace/{workspaceId}
description: Obtener todos los chatbots del workspace
---

<ParamField path="workspaceId" type="number" required>
  ID del workspace (debe ser el propio workspace del usuario)
</ParamField>

<RequestExample>
```bash cURL
curl https://api.scrivot.cl/v1/chatbot/workspace/7 \
  -H "Authorization: Bearer <service_token>"
```
</RequestExample>
